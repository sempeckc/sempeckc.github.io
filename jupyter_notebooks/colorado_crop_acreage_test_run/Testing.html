Okie doke, so let's see what we can do with some data. We're going to look at some public data linked from Data.gov, called CropScape, which shows the acreage of various agricultural vegetation grown in areas all around the US. I specifically localized the dataset to Colorado, and gathered all the crop acreage data for each year from 2008 to 2016. Let's start off by loading one of them to pandas and seeing what it looks like.

Okie dokie, so it looks like each of these data files contain: 1. A 'Category' for what crop we're talking about 2. A 'Value' integer corresponding to the specific 'Category' 3. A 'Count' for the number of pixels in the data (CropScape uses Geospatial coordinates, these pixels correspond to the number of pixels in the dataset which directly corresponds to the next value 'Acreage'. 4. An 'Acreage' for the number of acres that 'Category' represents in Colorado for that Year. Let's try and use the python module 'Seaborn' to show the column comparison of Category to Acreage and produce a pretty pretty graph!

Alright, so let's go with Tree crops. It won't be 8, but let's choose: Apples(46), Cherries(44), Peaches(38), and Other Tree Crops(45). That's 4. So let's make a list with our chosen categories by their (index) and grab out those categories.
